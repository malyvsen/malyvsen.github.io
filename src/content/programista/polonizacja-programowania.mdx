~~Natywne~~ rodzime zastępniki językowych nowotworów.

- _Refactor - remont_
- _Call - wywołać_
- _Call [a remote API] - stukać_
- _Return value - zwrotka_
- _Push - wypchnąć_ (nie tłumaczymy rzeczownika, aby uniknąć kwiatków w rodzaju _zrobić wypchnięcie_)
- _Pull - zaciągnąć_
- _Commit - zmianka, zmiankować_
- _Merge - zjednanie, zjednać_
- _Pull request - tiramisù_ (z włoskiego _tira - ciągnij, mi - mnie, sù - ku górze_)
- _To approve - klepnąć_
- _Main branch - pień_
- _Package - paczka_
- _Update - odnawiać_
- _Manager - ogarniacz_, np. _ogarniacz paczek_
- _Owner - pan/pani_
- _Upload - wrzutka, wrzucać_
- _Download - pobiór, pobierać_
- _Azure - Lazur_
- _Bottleneck - wąskie gardło_
- _To wrap - pakować_
- _Wrapper - opakowanie_
- _Ping - dźgnąć_
