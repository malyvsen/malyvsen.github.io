~~Natywne~~ rodzime zastępniki językowych nowotworów.

- *Refactor - remont*
- *Call - wywołać*
- *Call [a remote API] - stukać*
- *Return value - zwrotka*
- *Push - wypchnąć* (nie tłumaczymy rzeczownika, aby uniknąć kwiatków w rodzaju *zrobić wypchnięcie*)
- *Pull - zaciągnąć*
- *Commit - zmianka, zmiankować*
- *Merge - zjednanie, zjednać*
- *Pull request - tiramisù* (z włoskiego *tira - ciągnij, mi - mnie, sù - ku górze*)
- *To approve - klepnąć*
- *Main branch - pień*
- *Package - paczka*
- *Update - odnawiać*
- *Manager - ogarniacz*, np. *ogarniacz paczek*
- *Owner - pan/pani*
- *Upload - wrzutka, wrzucać*
- *Download - pobiór, pobierać*
- *Azure - Lazur*
- *Bottleneck - wąskie gardło*
- *To wrap - pakować*
- *Wrapper - opakowanie*
- *Ping - dźgnąć*
